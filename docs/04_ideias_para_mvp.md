# Ideias para o MVP real

## MVP 1 — Demo + login

- Next.js ou Astro.
- Supabase Auth.
- Usuários/personagens.
- Tela de sessão.
- Tela de segredos.
- Controle de visibilidade simples.

## MVP 2 — Captura e upload

- Upload de arquivos Craig.
- Upload de OBS backup.
- Upload/export do Roll20 chat.
- Storage privado.
- Metadados da sessão.

## MVP 3 — Transcrição

- Worker Docker com ffmpeg.
- Quebra de áudio.
- OpenAI transcription por faixa.
- Tabela transcript_segments.

## MVP 4 — Auditoria

- Classificação automática.
- Candidatos de canon.
- Candidatos de fala marcante.
- Candidatos de bastidores.
- Tela de aprovação.

## MVP 5 — Segredos

- Diário privado.
- Segredo com DM.
- Segredo compartilhado.
- Histórico de revelação.
- “Quem sabe o quê”.

## MVP 6 — Publicação

- Recap público.
- Recap privado.
- Timeline.
- Export Markdown/JSON opcional.
