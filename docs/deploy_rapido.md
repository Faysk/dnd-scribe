# Deploy rápido da demo

Esta demo é estática. Pode subir em qualquer lugar que sirva HTML, CSS e JS puro.

## Vercel

1. Criar um repositório no GitHub.
2. Copiar os arquivos desta pasta para o repositório.
3. Importar o projeto na Vercel.
4. Framework preset: `Other` ou `Static`.
5. Build command: deixar vazio.
6. Output directory: `.`.
7. Deploy.

## Netlify

1. Arrastar a pasta `dnd_scribe_demo` para o Netlify Drop; ou
2. Conectar o repositório GitHub.
3. Build command: vazio.
4. Publish directory: `.`.

## Cloudflare Pages

1. Conectar repo.
2. Framework preset: `None`.
3. Build command: vazio.
4. Output directory: `/` ou `.`.
