# Segredos e Conhecimento

O sistema separa duas coisas diferentes:

## 1. Quem vê no sistema

Controle técnico de acesso:

- somente dono;
- dono + DM;
- players escolhidos + DM;
- somente DM;
- mesa inteira;
- público.

## 2. Quem sabe na ficção

Controle narrativo:

- Dandelion sabe;
- Astel sabe;
- Screacky sabe;
- NPC sabe;
- povo sabe;
- ninguém sabe;
- é só conversa fora de personagem.

## Tipos de segredo

### Diário pessoal

Visível para o jogador dono e para o DM/lore admin.

Uso:

- pensamentos pessoais;
- rascunhos;
- teorias;
- ideias de músicas;
- anotações emocionais.

Não vira canon e não afeta o mundo automaticamente. O DM pode consultar para organizar a lore, mas qualquer consequência precisa de validação.

### Segredo de personagem

Visível para jogador + DM.

Uso:

- decisão secreta;
- descoberta privada;
- pacto;
- informação que pode gerar consequência.

Pode virar canon privado.

### Segredo compartilhado

Visível para um ou mais players + DM.

Uso:

- planos combinados;
- decisões entre personagens;
- informações parcialmente reveladas.

### Segredo do DM

Visível só para o DM.

Uso:

- planos de NPC;
- verdade oculta;
- armadilhas narrativas;
- lore ainda não revelada.

### Canon público

Visível para todos da campanha.

Uso:

- fatos aprovados;
- recap;
- eventos oficiais;
- timeline pública da mesa.

## Regra operacional

> Diário pessoal é rascunho. Segredo validado pelo DM é munição narrativa.
