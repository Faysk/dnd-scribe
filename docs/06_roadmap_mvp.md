# Roadmap MVP

## Fase 1 — Protótipo validado

- Mostrar demo visual para a mesa.
- Ajustar nomes, telas e regras de segredo.
- Definir o que realmente precisa no MVP.

## Fase 2 — Projeto real

- Next.js na Vercel.
- Supabase Auth com Google.
- Supabase Postgres com RLS.
- Upload básico de arquivos.
- Cadastro de sessão.
- Gestão de usuários.

## Fase 3 — Captura e ingestão

- Upload de Craig multitrack.
- Upload de OBS backup.
- Upload/export Roll20 chat.
- Parser de Craig notes.
- Parser de comandos `!dnd`.

## Fase 4 — Transcrição

- Worker Docker com Node, Python e ffmpeg.
- Split de áudio.
- Transcrição por faixa via OpenAI.
- Merge por timestamp.

## Fase 5 — IA e auditoria

- Classificação automática.
- Candidatos de canon.
- Candidatos de fala.
- Candidatos de bastidor.
- Detecção de possíveis segredos.
- Tela de revisão.

## Fase 6 — Publicação

- Recap curto.
- Recap completo.
- Mudanças de canon.
- Timeline.
- Bastidores aprovados.
- Páginas públicas/privadas.
