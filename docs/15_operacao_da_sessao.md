# 15 — Operação da Sessão

## Checklist antes da sessão

```txt
[ ] Criar sessão no DnD Scribe
[ ] Confirmar participantes/personagens
[ ] Confirmar consentimento de gravação
[ ] Abrir Roll20
[ ] Ativar Roll20 Logger
[ ] Entrar no Discord
[ ] Iniciar Craig
[ ] Iniciar OBS
[ ] Fazer teste de áudio
[ ] Fazer SYNC DnD no áudio/Craig/Roll20
[ ] Conferir se o Craig está gravando todos
```

## Script de início

Alguém fala no começo:

```txt
SYNC DnD. Sessão iniciada oficialmente.
```

No Roll20:

```txt
!ys sync Sessão iniciada oficialmente
!ys start 2026-06-27_sessao-XX
```

No Craig:

```txt
/note SYNC: sessão iniciada oficialmente
```

## Durante a sessão

Usar marcadores simples:

```txt
CANON: fato importante
FALA: fala marcante
BASTIDOR: piada boa
CORTAR: conversa aleatória
DUVIDA: revisar depois
CENA: nova cena
COMBATE: início/fim
NPC: novo NPC
ITEM: item novo
GANCHO: gancho futuro
```

## Checklist depois da sessão

```txt
[ ] Encerrar Craig
[ ] Salvar/baixar arquivos Craig
[ ] Encerrar OBS
[ ] Salvar OBS backup
[ ] Exportar Roll20 chat
[ ] Subir arquivos no DnD Scribe
[ ] Associar faixas aos jogadores/personagens
[ ] Enfileirar processamento
[ ] Conferir se worker iniciou
```

## Checklist de revisão

```txt
[ ] Revisar candidatos de canon
[ ] Rejeitar piadas marcadas errado
[ ] Aprovar falas marcantes
[ ] Marcar bastidores privados/públicos
[ ] Corrigir nomes próprios
[ ] Corrigir speaker/personagem
[ ] Revisar ganchos
[ ] Gerar recap curto
[ ] Gerar recap completo
[ ] Publicar versão aprovada
```

## Funções na mesa

### Mestre

- aprova canon;
- define segredo/público;
- revisa retcon;
- aprova publicação final.

### Jogadores

- marcam falas e bastidores;
- corrigem personagem/speaker;
- aprovam bastidores envolvendo eles;
- ajudam a revisar interpretação.

### Arquivista da sessão

Pode ser uma função rotativa:

- garante Craig/OBS;
- marca eventos;
- sobe arquivos;
- inicia processamento.

## Regra prática

Durante a sessão, marcar sem pensar muito.
Depois da sessão, revisar com calma.

Marcador falso é barato. Evento esquecido é caro. Igual comprar item mágico sem ler descrição: parece barato até explodir no bolso.
