# DnD Scribe — Visão Geral

O DnD Scribe é um sistema para registrar sessões longas de D&D com foco em memória auditável, transcrição, canonização e controle de segredos.

A ideia não é apenas guardar resumo. O sistema deve registrar:

- o que aconteceu;
- quem falou;
- quando aconteceu;
- qual fonte prova aquilo;
- quem pode ver no sistema;
- quem sabe dentro da ficção;
- se é canon público, canon privado, segredo, bastidor ou diário pessoal;
- se pode ser publicado.
