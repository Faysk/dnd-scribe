1. Instalar Mod/API no Roll20

Abra a pagina do jogo no Roll20, va em Settings -> Mod (API) Scripts -> New Script e cole o script abaixo.

Abrir script Mod
2. Instalar extensao Chrome local

O Roll20 bloqueia scripts externos por CSP. Use a extensao local do projeto em vez de bookmarklet.

Ver manifest

No Chrome, abra chrome://extensions, ative Developer mode, clique Load unpacked e selecione D:\Projects\dnd\integrations\roll20\chrome-extension.

3. Rodar na mesa

Abra o jogo no Roll20. Primeiro pause com !dndscribe off. Quando o painel pequeno da extensao aparecer, clique em Config e informe URL, campaign slug, sourceSessionId selecionado acima e token. Depois ligue com !dndscribe on.

Comandos no chat Roll20

!dndscribe status mostra o estado. !dndscribe off pausa. !dndscribe on religa. Se aparecer pacote grande DND_SCRIBE_EVENT no chat do GM, pause, recarregue a mesa e confira a extensao.

Seguranca

A API aceita somente `ROLL20_BRIDGE_TOKEN`. Sem esse token, a rota responde erro e nada e gravado.
